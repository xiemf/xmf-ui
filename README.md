# xmf-ui

## Project use

```
npm install --save-dev xmf-ui
import XmfUI from 'xmf-ui'
import 'xmf-ui/lib/xmf-ui.css'
Vue.use(XmfUI)
```

## Components

### Affix

```
<x-affix></x-affix>
在被包裹组件在视图中消失时浮出 适用于浮动Menu
```

### Icon

```
<x-icon><x-icon>
图标组件 扩充ing
```

### Avatar

```
<x-avatar></avatar>
头像组件 支持url x-icon defaultSlot
```

### Input

```
<x-input />
输入框组件
```
